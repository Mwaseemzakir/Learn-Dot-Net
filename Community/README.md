# .NET Community & Forums

Places to connect with other .NET developers, ask questions, and share knowledge.

## Forums & Q&A

1. [Stack Overflow — .NET](https://stackoverflow.com/questions/tagged/.net)

	The most popular Q&A site for programmers. Use the `.net`, `c#`, and `asp.net-core` tags to find and ask questions.

2. [Reddit r/dotnet](https://www.reddit.com/r/dotnet/)

	Active subreddit for .NET news, discussions, and questions with over 100k members.

3. [Reddit r/csharp](https://www.reddit.com/r/csharp/)

	Dedicated C# subreddit for language-specific discussions and help.

## Discord Servers

1. [C# Discord](https://discord.gg/csharp)

	One of the largest active Discord communities for C# developers with dedicated help channels for beginners and experts alike.

2. [.NET Community Discord](https://discord.gg/dotnet)

	Official .NET Foundation Discord server with channels for various .NET topics and sub-communities.

## GitHub Discussions

1. [.NET Runtime Discussions](https://github.com/dotnet/runtime/discussions)

	Official discussions for the .NET runtime — great for understanding design decisions and future plans.

2. [ASP.NET Core Discussions](https://github.com/dotnet/aspnetcore/discussions)

	Official discussions for ASP.NET Core features, proposals, and community questions.

## Social Media

1. [Twitter/X — #dotnet](https://twitter.com/search?q=%23dotnet)

	Follow the `#dotnet` and `#csharp` hashtags for daily news, tips, and content from the community.

2. [LinkedIn — .NET Developers](https://www.linkedin.com/groups/40961/)

	Professional networking group for .NET developers.

## Organizations

1. [.NET Foundation](https://dotnetfoundation.org/)

	Independent non-profit organization that supports the .NET ecosystem and open source projects.

2. [Microsoft Learn Community](https://learn.microsoft.com/en-us/community/)

	Official Microsoft community hub for Q&A, events, and learning resources.
