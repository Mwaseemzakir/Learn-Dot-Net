# .NET & C# Conferences

A list of major conferences where .NET and C# developers can learn, network, and grow.

## Annual Conferences

1. [.NET Conf](https://www.dotnetconf.net/)

	A free, virtual, community event organized by Microsoft and the .NET community. Covers the latest features in .NET, C#, ASP.NET, and more. Held every November alongside the .NET release.

2. [NDC Conferences](https://ndcconferences.com/)

	NDC (Norwegian Developers Conference) holds events in Oslo, London, Sydney, and Porto. One of the best software conferences with deep .NET, architecture, and engineering content.

3. [Microsoft Build](https://build.microsoft.com/)

	Microsoft's annual developer conference featuring announcements and sessions on .NET, Azure, AI, and all Microsoft technologies.

4. [CodeMash](https://www.codemash.org/)

	A multi-disciplinary technology conference held in Sandusky, Ohio. Strong .NET track alongside sessions on web, mobile, cloud, and more.

5. [DotNext](https://dotnext.ru/)

	A conference dedicated exclusively to .NET and related technologies. Covers performance, internals, new language features, and real-world practices.

6. [Techorama](https://www.techorama.be/)

	A technology conference in Belgium and the Netherlands with excellent .NET, Azure, and Microsoft ecosystem coverage.

7. [Update Conference Prague](https://www.updateconference.net/)

	A Microsoft-focused developer conference in Prague with strong .NET and Azure content.

## Online / On-Demand

1. [JetBrains .NET Days Online](https://www.jetbrains.com/dotnet/)

	Free online conference by JetBrains focused on .NET development tools, Rider, and best practices.

2. [Microsoft Learn Events](https://learn.microsoft.com/en-us/events/)

	On-demand and live sessions directly from Microsoft on all .NET topics, available year-round.
