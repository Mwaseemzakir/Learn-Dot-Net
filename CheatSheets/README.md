# .NET & C# Cheat Sheets

Quick reference guides to keep open while coding.

## C# Language

1. [C# Language Reference — Microsoft Docs](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/)

	The official and most complete reference for all C# syntax and language features.

2. [What's New in C# — Version History](https://learn.microsoft.com/en-us/dotnet/csharp/whats-new/)

	Summary of new features introduced in each C# version — essential for staying current.

3. [C# Cheat Sheet — QuickRef.me](https://quickref.me/cs.html)

	A concise single-page reference for C# syntax, types, and common patterns.

## LINQ

1. [101 LINQ Samples — Microsoft](https://learn.microsoft.com/en-us/samples/dotnet/try-samples/101-linq-samples/)

	Interactive 101-sample tour covering every LINQ operator with runnable examples.

2. [Enumerable API Reference](https://learn.microsoft.com/en-us/dotnet/api/system.linq.enumerable)

	Full API reference for all LINQ extension methods on `IEnumerable<T>`.

## Async / Await

1. [Async/Await Best Practices — Stephen Cleary](https://learn.microsoft.com/en-us/archive/msdn-magazine/2013/march/async-await-best-practices-in-asynchronous-programming)

	The definitive guide to async best practices in C#. A must-read before writing production async code.

2. [ConfigureAwait FAQ — Stephen Toub](https://devblogs.microsoft.com/dotnet/configureawait-faq/)

	Deep dive into `ConfigureAwait` by a core .NET team member. Explains every nuance.

## Entity Framework Core

1. [EF Core Querying Docs](https://learn.microsoft.com/en-us/ef/core/querying/)

	Official docs for all EF Core querying patterns — loading strategies, filtering, and tracking.

2. [EF Core Migrations Reference](https://learn.microsoft.com/en-us/ef/core/managing-schemas/migrations/)

	Step-by-step guide to managing database schema changes with EF Core migrations.
