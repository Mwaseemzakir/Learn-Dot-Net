## Role-Based .NET Learning Paths

### Junior .NET Developer

Master the fundamentals before moving to frameworks.

1. **C# Basics** — Variables, types, control flow, methods, and classes
2. **OOP Concepts** — Inheritance, polymorphism, encapsulation, and abstraction
3. **Collections & LINQ** — List, Dictionary, IEnumerable, and LINQ operators
4. **Async / Await** — Tasks, async patterns, and CancellationToken
5. **ASP.NET Core Basics** — Web API, routing, controllers, and middleware pipeline
6. **Entity Framework Core** — Models, DbContext, migrations, and basic CRUD
7. **SQL Basics** — SELECT, JOINs, indexes, and stored procedures
8. **Git** — Commits, branches, pull requests, merge vs rebase
9. **Unit Testing** — xUnit or NUnit, Arrange-Act-Assert pattern

---

### Mid-Level .NET Developer

Build on the fundamentals with architecture and advanced patterns.

1. **Design Patterns** — Repository, Factory, Decorator, Strategy, and Observer
2. **SOLID Principles** — Apply each principle with real-world C# examples
3. **Clean Architecture** — Layers, dependency rule, use cases, and abstractions
4. **CQRS & MediatR** — Commands, queries, handlers, and pipeline behaviors
5. **REST API Design** — Versioning, pagination, error handling, and OpenAPI
6. **Authentication & Authorization** — JWT, OAuth 2.0, and ASP.NET Core Identity
7. **Caching Strategies** — In-memory cache, Redis, and cache invalidation
8. **Docker** — Containerizing .NET apps, Dockerfile best practices, docker-compose
9. **CI/CD** — GitHub Actions or Azure DevOps pipelines for .NET
10. **Integration Testing** — WebApplicationFactory and TestContainers

---

### Senior .NET Developer

Advanced topics for building scalable, observable, and resilient systems.

1. **Microservices Architecture** — Service decomposition, bounded contexts, and API gateways
2. **Event-Driven Architecture** — MassTransit, RabbitMQ, and the outbox pattern
3. **Performance Engineering** — BenchmarkDotNet, Span<T>, memory profiling, and allocation reduction
4. **Observability** — OpenTelemetry, distributed tracing, metrics, and dashboards
5. **Cloud-Native .NET** — Azure App Services, AKS, Azure Service Bus, and .NET Aspire
6. **Event Sourcing** — Event stores, Marten, projections, and CQRS with events
7. **Security Engineering** — Threat modeling, OWASP Top 10 for .NET, and secrets management
8. **System Design** — Scalability patterns, fault tolerance, CAP theorem, and eventual consistency
