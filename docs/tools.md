# Tools for Developers

## Choosing the right tool

| **Purpose** | **Tool** | **Description** |
| --- | --- | --- |
| **IDE** | [Visual Studio Community](https://visualstudio.microsoft.com/vs/community/) | Full-featured free IDE for .NET development |
| | [Visual Studio Code](https://code.visualstudio.com/) | Lightweight, extensible code editor |
| | [JetBrains Rider](https://www.jetbrains.com/rider/) | Cross-platform .NET IDE with excellent refactoring |
| **Source Control** | [Git](https://git-scm.com/) | Distributed version control system |
| | [GitHub Desktop](https://desktop.github.com/) | GUI client for Git and GitHub |
| **Database** | [Microsoft SQL Server](https://www.microsoft.com/en-us/sql-server/) | Microsoft's relational database platform |
| | [Azure Data Studio](https://azure.microsoft.com/en-us/products/data-studio/) | Cross-platform database tool for SQL Server and Azure |
| | [DBeaver](https://dbeaver.io/) | Universal database tool supporting many databases |
| **API Testing** | [Postman](https://www.postman.com/) | API development and testing platform |
| | [Bruno](https://www.usebruno.com/) | Fast, open-source API client |
| | [Insomnia](https://insomnia.rest/) | Collaborative API design and testing tool |
| **Profiling & Diagnostics** | [PerfView](https://github.com/microsoft/perfview) | CPU and memory performance analysis tool by Microsoft |
| | [Visual Studio Profiler](https://learn.microsoft.com/en-us/visualstudio/profiling/) | Built-in profiling tools inside Visual Studio |
| | [dotTrace](https://www.jetbrains.com/profiler/) | JetBrains .NET performance profiler |
| | [dotMemory](https://www.jetbrains.com/dotmemory/) | JetBrains .NET memory profiler |
| | [dotnet-counters](https://learn.microsoft.com/en-us/dotnet/core/diagnostics/dotnet-counters) | CLI tool for live performance counter monitoring |
| **Load & Stress Testing** | [Apache JMeter](https://jmeter.apache.org/) | Open-source load testing tool |
| | [k6](https://k6.io/) | Modern, scriptable load testing tool |
| | [NBomber](https://nbomber.com/) | .NET-native load testing framework |
| | [Gatling](https://gatling.io/) | High-performance load testing tool |
| | [Locust](https://locust.io/) | Scalable, Python-based load testing |
| | [West Wind WebSurge](https://websurge.west-wind.com/) | .NET-focused HTTP load testing tool |
| **Containerization** | [Docker Desktop](https://www.docker.com/products/docker-desktop/) | Container development environment |
| | [Podman](https://podman.io/) | Daemonless container engine, Docker-compatible |
| **CI/CD** | [GitHub Actions](https://github.com/features/actions) | Workflow automation built into GitHub |
| | [Azure DevOps](https://azure.microsoft.com/en-us/products/devops/) | Microsoft's complete DevOps platform |
| **Cloud** | [Azure CLI](https://learn.microsoft.com/en-us/cli/azure/) | Command-line tool for managing Azure resources |
| | [Azure Portal](https://portal.azure.com/) | Web-based management console for Azure |
