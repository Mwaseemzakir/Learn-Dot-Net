# List of .NET Podcasts

1. [.NET Rocks](https://www.dotnetrocks.com/)

	The original .NET podcast with Carl Franklin and Richard Campbell. Over 1900 episodes covering every corner of the .NET ecosystem, from language internals to career advice.

2. [Merge Conflict](https://www.mergeconflict.fm/)

	Weekly discussion with Frank Krueger and James Montemagno on .NET, MAUI, mobile development, and the broader world of software.

3. [No Dogma Podcast](https://nodogmapodcast.bryanhogan.net/)

	Bryan Hogan interviews .NET developers and architects about real-world software development, design decisions, and career growth.

4. [The .NET Core Podcast](https://dotnetcore.show/)

	Dedicated exclusively to .NET Core and .NET 5+ topics. Interviews with community members building real-world .NET applications.

5. [Coding Blocks](https://www.codingblocks.net/)

	Discussion-format podcast covering software development best practices, design patterns, architecture, and developer productivity.

6. [Adventures in .NET](https://topenddevs.com/podcasts/adventures-in-net)

	Panel discussions and interviews covering .NET topics from web APIs to desktop development to cloud services.

7. [Software Engineering Radio](https://www.se-radio.net/)

	Long-form interviews with senior software engineers on architecture, distributed systems, and engineering practices.

8. [Programming Throwdown](https://www.programmingthrowdown.com/)

	Each episode covers a different programming language or technology, regularly featuring .NET and C# topics.
