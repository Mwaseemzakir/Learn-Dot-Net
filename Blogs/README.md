# List of Blogging Sites Creating Content on C# and .NET

Millions of websites out there — at the end of the day we like few because we understand their style of teaching.

Putting down some good blogging sites along with famous bloggers. Go read and find out whose writing rings the bell in your mind. :smile:


1. [Code4IT](https://www.code4it.dev/)

	Code4IT covers almost all topics from basics to advanced in C# — great for building a solid foundation.

2. [Andrew Lock](https://andrewlock.net/)

	Andrew is famous for his ASP.NET Core in Action book and produces in-depth articles on all areas of C# and .NET.

3. [Code Maze](https://code-maze.com/)

	If someone asks me which blogging site is my favorite, I will vote for CodeMaze. You will find each and everything here that one can imagine in the .NET world.

4. [.NET Tricks](https://www.dotnettricks.com/)

	This site contains books, courses, and talks on .NET and other languages as well.

5. [Method Poet](https://methodpoet.com/)

	With 10+ years of experience, Kristijan talks on architecture-level topics and software design.

6. [Josip Miskovic](https://josipmisko.com/)

	Practical tutorials on C# and .NET fundamentals and web development.

7. [.NET Tutorials](https://dotnettutorials.net/)

	MVC, C#, Web API, Design Patterns, and .NET — additionally covers other languages like Java, Python, C, and C++.

8. [Code with Mukesh](https://codewithmukesh.com)

	Mukesh talks about Azure, AWS, C#, and .NET with a focus on practical, hands-on tutorials.

9. [Steve Smith (Ardalis)](https://ardalis.com/blog/)

	Quite famous in the .NET world — his book is featured on the Microsoft site. Focuses on architecture, clean code, and design patterns.

10. [Coding Sonata](https://codingsonata.com)

	Talks about C# and .NET with clear, well-structured articles.

11. [Dave Callan](https://davecallan.com)

	Dave is known for benchmarking in C# and useful Visual Studio tips and tricks.

12. [Milan Jovanovic](https://www.milanjovanovic.tech/)

	Milan talks on advanced topics of C# and .NET — Clean Architecture, CQRS, and software design.

13. [Julio Casal](https://juliocasal.com/)

	With 10+ years of experience, Julio covers both basic and advanced .NET topics including Docker and microservices.

14. [Stefan Djokic](https://stefandjokic.tech/)

	Talks on basic and advanced topics of C#/.NET and design patterns.

15. [Stevan Giesel](https://steven-giesel.com/)

	This developer from Zurich will take you into the advanced world of .NET — he loves to play with code.

16. [Steve Gordon](https://www.stevejgordon.co.uk/)

	Microsoft MVP, Pluralsight author, and senior engineer at Elastic. Passionate about ASP.NET Core internals, performance, and HTTP.

17. [Dan Patrascu-Baba](https://danpatrascu.com/)

	Developer and .NET trainer focused on Microsoft Cloud, Office 365, and Azure. Keeps you up to date with the latest Microsoft services.

18. [Code to Solutions](https://codetosolutions.com/)

	Covers ASP.NET Core, Web API, MVC, C#, Entity Framework, Angular, AWS, and SQL Server with practical coding examples.

19. [Stephen Cleary](https://blog.stephencleary.com/)

	Microsoft MVP and author of a book on concurrency, asynchronous, parallel, and multi-threading programming. An essential read for async code.

20. [Elmah.io Blog](https://blog.elmah.io/)

	Elmah.io is an error logging service for .NET web applications. Their blog covers .NET, C#, and web development topics.

21. [Code Guru](https://www.codeguru.com/)

	Covers Microsoft-related software development, mobile, database, and web application programming in C# and .NET.

22. [Rahul Nath](https://www.rahulpnath.com/blog/)

	Coder, YouTuber, and blogger. Talks about C#, .NET, and Azure in a clear and practical style.

23. [Rmauro Dev](https://rmauro.dev)

	Your go-to source for insights, tips, and tricks on building robust, scalable, and secure web applications using C# and Azure.

24. [Scott Hanselman](https://www.hanselman.com/blog/)

	Microsoft Partner Program Manager and developer advocate. Covers .NET, developer tools, productivity, and technology broadly.

25. [Mark Seemann (ploeh)](https://blog.ploeh.dk/)

	Expert on functional programming in C#, dependency injection patterns, TDD, and software design. Deeply thoughtful long-form articles.

26. [Khalid Abuhakmeh](https://khalidabuhakmeh.com/)

	JetBrains developer advocate writing about .NET, Rider, practical development tips, and open source projects.
