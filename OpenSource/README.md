# .NET Open Source Projects

Notable open source .NET projects to learn from, use, or contribute to.

## Microsoft Official

| Project | Description | GitHub |
| --- | --- | --- |
| ASP.NET Core | Web framework for building modern web apps and APIs | [dotnet/aspnetcore](https://github.com/dotnet/aspnetcore) |
| Entity Framework Core | Modern object-database mapper for .NET | [dotnet/efcore](https://github.com/dotnet/efcore) |
| .NET Runtime | Core runtime, base class libraries, and tools | [dotnet/runtime](https://github.com/dotnet/runtime) |
| .NET MAUI | Cross-platform UI app framework | [dotnet/maui](https://github.com/dotnet/maui) |
| BenchmarkDotNet | Powerful micro-benchmarking library | [dotnet/BenchmarkDotNet](https://github.com/dotnet/BenchmarkDotNet) |
| Roslyn | .NET compiler platform (C# and VB.NET) | [dotnet/roslyn](https://github.com/dotnet/roslyn) |
| .NET Aspire | Cloud-ready stack for building distributed apps | [dotnet/aspire](https://github.com/dotnet/aspire) |
